Frequently asked

  • The alarm didn't ring. iOS aggressively kills background apps. Make sure Notifications are allowed for WakeFlow (Settings → Notifications → WakeFlow) and that Focus / Do Not Disturb isn't blocking it.
  • AI Coach says "Set up your Gemini API key." WakeFlow uses Google's Gemini API for the coaching summaries. You can grab a free key from aistudio.google.com and paste it into Settings → AI Coach.
  • How do I change the app language? Settings → Language. You can pick a language manually or let it follow your iOS system language.
  • What happens to my data if I delete the app? All sleep records, routines, and your API key are stored locally on your device. Deleting the app erases everything — we don't keep a copy on any server.
  • Can I remove ads? Watch a rewarded ad from Settings to silence banners and interstitials for 30 minutes. A permanent option will arrive in a future update.

Preguntas frecuentes

  • La alarma no sonó. iOS cierra las apps en segundo plano. Asegúrate de tener notificaciones activadas para WakeFlow (Ajustes → Notificaciones → WakeFlow) y que el modo Concentración / No Molestar no esté bloqueándolas.
  • El Coach de IA dice "Configura tu clave API de Gemini". WakeFlow usa la API de Gemini de Google para los resúmenes. Puedes obtener una clave gratuita en aistudio.google.com y pegarla en Ajustes → Coach de IA.
  • ¿Cómo cambio el idioma? Ajustes → Idioma. Puedes elegir un idioma manualmente o seguir el idioma del sistema.
  • ¿Qué pasa con mis datos si borro la app? Los registros de sueño, rutinas y tu clave API se guardan localmente en tu dispositivo. Al borrar la app se elimina todo — no guardamos copia en ningún servidor.
  • ¿Puedo quitar los anuncios? Mira un anuncio recompensado desde Ajustes para silenciar banners e intersticiales durante 30 minutos. Una opción permanente llegará en futuras actualizaciones.

자주 묻는 질문

  • 알람이 울리지 않아요. iOS는 백그라운드 앱을 적극적으로 종료해요. 설정 → 알림 → WakeFlow 에서 알림이 켜져 있는지, 집중 모드/방해금지 모드가 알람을 막고 있지 않은지 확인해 주세요.
  • AI 코치가 "Gemini API 키를 설정해주세요"라고 나와요. AI 코치는 Google Gemini API를 사용해요. aistudio.google.com 에서 무료로 키를 발급받아 설정 → AI 코치 에 붙여넣으면 돼요.
  • 앱 언어를 어떻게 바꾸나요? 설정 → 언어 에서 직접 고르거나, 시스템 언어를 그대로 따라가게 할 수 있어요.
  • 앱을 삭제하면 제 데이터는 어떻게 되나요? 모든 수면 기록과 루틴, API 키는 기기 안에만 저장돼요. 앱을 삭제하면 모두 함께 사라지고, 저희 서버에는 어떤 사본도 남지 않아요.
  • 광고를 제거할 수 있나요? 설정 화면에서 보상형 광고를 시청하면 30분 동안 배너와 전면 광고가 사라져요. 영구 제거 옵션은 추후 업데이트에서 추가할 예정이에요.

WakeFlowをご利用いただきありがとうございます。小さなインディーチームが作っているアプリです。うまく動かないことやアイデアがあれば、ぜひお知らせください。

お問い合わせ

Email: smchoi@clify.co.kr

通常2営業日以内に返信します。iOSのバージョン、端末モデル、状況の簡単な説明を添えてお送りいただけると助かります。

よくある質問

  • アラームが鳴りません。 iOSはバックグラウンドのアプリを積極的に終了します。設定 → 通知 → WakeFlow で通知が許可されているか、集中モード／おやすみモードが妨げていないかをご確認ください。
  • AIコーチに「Gemini APIキーを設定してください」と表示されます。 AIコーチはGoogleのGemini APIを利用します。aistudio.google.com から無料でキーを発行し、設定 → AIコーチ に貼り付けてください。
  • アプリの言語はどう変えますか? 設定 → 言語 から手動で選ぶか、システム言語に合わせて自動切替できます。
  • アプリを削除するとデータはどうなりますか? 睡眠記録、ルーティン、APIキーはすべて端末内にのみ保存されます。アプリを削除するとすべて消え、当方サーバーに控えは残りません。
  • 広告を消せますか? 設定でリワード広告を視聴すると、30分間バナー・インタースティシャル広告を非表示にできます。恒久的なオプションは今後のアップデートで提供予定です。

支持中心

感谢使用 WakeFlow。我们是一个小型独立团队，如果遇到问题或有想法，欢迎随时告诉我们。

联系方式

邮箱: smchoi@clify.co.kr

通常会在 2 个工作日内回复。请附上你的 iOS 版本、设备型号以及问题的简要描述。

常见问题

  • 闹钟没有响。 iOS 会主动结束后台应用。请确认在 设置 → 通知 → WakeFlow 中允许通知，并确认专注模式 / 勿扰模式没有屏蔽闹钟。
  • AI 教练提示"请设置 Gemini API 密钥"。 WakeFlow 使用 Google Gemini API 生成分析。请在 aistudio.google.com 免费申请密钥，并粘贴到 设置 → AI 教练。
  • 如何切换应用语言? 在 设置 → 语言 中手动选择，或跟随系统语言。
  • 删除应用后我的数据会怎样? 所有睡眠记录、例程和 API 密钥仅保存在你的设备上。删除应用后全部消失，我们的服务器不保留任何副本。
  • 能去除广告吗? 在设置中观看激励视频广告即可在 30 分钟内隐藏横幅和插页广告。永久去广告选项将在未来版本提供。
