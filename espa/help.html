Help & Support

Getting Started

Tap the home tab and pick "Continue learning" to start a 12-card session. Each card lets you reveal the meaning, hear the Spanish word, and read example sentences.

Hearts

Wrong answers cost a heart. When hearts hit zero you can refill from the empty-hearts screen — take a quick break and the lesson stays available.

Mascots

Choose a study buddy from the Profile tab. PRO mascots can be unlocked by enabling the PRO demo from the unlock dialog.

Daily Goal

Set your daily word goal in Settings. The home tab shows how many words are left to hit your goal today.

Languages

Use Settings → Language to switch the entire UI between English, Korean, Japanese, and Chinese.

도움말

시작하기

홈 탭에서 "학습 계속하기"를 탭하면 12장의 카드 세션이 시작됩니다. 각 카드에서 의미 보기 → 발음 듣기 → 예문 확인을 할 수 있습니다.

하트

오답 시 하트가 차감됩니다. 하트가 0이 되면 회복 화면에서 즉시 회복하거나 잠시 휴식 후 다시 도전할 수 있습니다.

마스코트

프로필 탭에서 학습 친구를 선택할 수 있습니다. PRO 마스코트는 잠금해제 대화상자에서 PRO 데모를 켜면 사용 가능합니다.

일일 목표

설정에서 일일 단어 목표를 조정하세요. 홈 탭에서 오늘 남은 단어 수를 확인할 수 있습니다.

언어

설정 → 학습 / 번역 언어에서 한국어, 영어, 일본어, 중국어로 UI 전체를 즉시 전환할 수 있습니다.

ヘルプ

はじめに

ホームタブの「学習を続ける」をタップすると、12枚カードのセッションが始まります。各カードで意味の表示 → 発音 → 例文の確認ができます。

ハート

間違えるとハートが減ります。0になったら回復画面ですぐ回復するか、少し休んでから再挑戦できます。

マスコット

プロフィールタブで学習相棒を選べます。PROマスコットは、ロック解除ダイアログで「PROデモを有効化」を選ぶと使えます。

毎日の目標

設定で1日の学習単語数を調整できます。ホームタブには、目標達成まであと何単語かが表示されます。

言語

設定 → 学習 / 翻訳言語から、UIを日本語・英語・韓国語・中国語に瞬時に切り替えられます。

帮助

开始使用

点击主页选项卡的「继续学习」即可开始 12 张卡片的学习会话。每张卡片可依次:看意思 → 听发音 → 看例句。

爱心

答错会扣除一颗爱心。归零后可在「爱心耗尽」页面立即恢复,或稍作休息后再来。

吉祥物

在「个人」选项卡选择学习伙伴。PRO 吉祥物可在解锁对话框中开启「PRO 演示」后使用。

每日目标

在设置中调整每日学习的单词数量。主页会显示今天距离目标还差多少。

语言

设置 → 学习 / 翻译语言可在中文、英文、韩文、日文之间瞬时切换整个 UI。
